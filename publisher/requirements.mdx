## System

- WordPress Version: 4.9.6 or higher
- PHP Version: 7.0 or higher

## PHP Modules

- curl
- iconv
- SimpleXML
- OpenSSL
- Exif
- zlib
- mbstring

## Compatible Hosters*

- [Uberspace](https://uberspace.de/)
- [Hetzner Managed Hosting](https://www.hetzner.com/webhosting)
- [All Inklusive Webhosting](https://all-inkl.com/webhosting)
- [Hosteurope Webhosting](https://www.hosteurope.de/webhosting-loesungen/)
- Dedicated Server/vServer solutions with installed PHP modules
- Docker environments running the [official wordpress image](https://hub.docker.com/_/wordpress)

_* Incomplete list, feel free to add further working options_
