Podlove Web Player

The fast, flexible and responsive podcast player powered by podlove meta data.

Podlove Web Player is a HTML5 based web player for audio and video media files that is optimized and extended for the specific needs of podcasters. This player integrates well with the Podlove Podcast Publisher but can also be used independent from it.

Made for Podcasters​

  • Podcast Chapters
  • Transcripts
  • Sharing & Embedding
  • Download integrated
  • Audio Controls, including playback speed
  • Supports multiple audio file formats
  • Supports live streams
